weatherOrNot

A scheduler app in which a user creates an account, and inputs events into their own calendar. These events are only visible to the user's specific login profile. Also the calendar provides the weather for the next 5 calendar days, starting from today's date.

My Role:

I dabbled in all areas of execution for this app; both back-end and front-end. I assisted in creating the sequelize models for the database. Assisted in the styling of both the login/sign-in page and the scheduler/calendar page. Implemented all of the fetch calls and applied them to the calendar. Created all of the routes/endpoints on the back-end. Integrated the 3rd party API weather function and created the logic to have it give the weather for 5 calendar days.
